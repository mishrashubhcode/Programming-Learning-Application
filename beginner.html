Basic Concepts of Python

Introduction

Python is a high-level, interpreted programming language that was first released in 1991. It is designed to be easy to read and write, and is known for its clear and concise syntax. Python is an object-oriented language, which means that it is built around the concept of objects, which can contain data and code.

Data Types

Python has several built-in data types, including:

  • Numbers (integers, floats, and complex numbers)
  • Strings
  • Lists
  • Tuples
  • Dictionaries
  • Sets
  • Booleans
  • None

Variables

In Python, a variable is a name that refers to a value. Variables are created when they are first assigned a value. Python uses dynamic typing, which means that a variable can refer to a value of any type. Variable names are case-sensitive.

Operators

Python has several types of operators, including:

  • Arithmetic operators (+, -, *, /, %, //, **)
  • Comparison operators (==, !=, >, <, >=, <=)
  • Assignment operators (=, +=, -=, *=, /=, %=, //=, **=)
  • Logical operators (and, or, not)
  • Identity operators (is, is not)
  • Membership operators (in, not in)
  • Bitwise operators (&, |, ^, ~, <<, >>)

Control Flow

Python has several types of control flow statements, including:

  • If statements
  • For loops
  • While loops
  • Try/except statements

Functions

A function is a block of code that performs a specific task. Functions can take parameters as input, and can return a value as output. In Python, functions are defined using the def keyword.

Modules

A module is a file that contains Python code. Modules can be imported into other Python code using the import statement. Python has a large standard library of modules that can be used for a variety of tasks.
